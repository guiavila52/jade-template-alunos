# Squad de Agentes IA — Template

O time de agentes de IA que você precisa pra operar seu negócio digital. Um COO (Chief Operating Officer) que comanda o squad, despacha tarefas pros agentes certos e garante que tudo roda como deve.

## O que você vai ter

- **Time completo de agentes prontos** — COO (Jade), copywriter, desenvolvedor front-end, designer, analista financeiro, contador, gestor de tráfego, customer success e mais.
- **Skills canônicas pra produção** — comandos prontos pra criar newsletter, página de vendas, carrossel, criativo de tráfego, relatório fiscal, análise de métricas e todo o resto.
- **Hooks de segurança ativos** — agentes bloqueantes que garantem que nada sai errado (revisão visual obrigatória antes de publicar, skill canônica antes de produzir, bateria de testes externa antes de entregar).
- **Estrutura do Segundo Cérebro** — knowledge organizado sobre identidade do seu negócio, tom de voz, produtos, ICP, histórias, ofertas e operação.

## Como instalar

1. Clique em **Code → Download ZIP** aqui no GitHub
2. Extraia a pasta em qualquer lugar no seu computador
3. Abra o **Claude Code Desktop** → clique em **Select folder** → aponte pra pasta extraída
4. Mande um `oi`

A Jade detecta que é a primeira vez e já cuida de tudo — vai se apresentar, listar os times disponíveis e pedir as informações do seu negócio pra montar o Segundo Cérebro. Você não precisa rodar nenhum comando.

## Como usar no dia a dia

Depois de configurado, sempre que quiser começar uma nova sessão com a Jade, mande:

```
/jade-iniciar
```

Ela verifica o que tem na lista de tarefas, apresenta as top 5, sugere a mais urgente e pergunta se pode começar. Você só precisa dizer "pode" — ela orquestra o resto.
