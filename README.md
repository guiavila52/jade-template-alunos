# Squad de Agentes — Template

Sistema operacional de agentes de IA para negócios digitais.

## Como começar

1. Clone este repositório
2. Abra no Claude Code
3. Execute `/configurar-squad` e responda as perguntas sobre seu negócio
4. Sua Jade estará pronta para operar

## Estrutura

```
squad/
├── CLAUDE.md        — framework e regras do squad
├── AGENTS.md        — 18 regras invioláveis
├── IDENTIDADE.md    — seus dados (preenchido via /configurar-squad)
├── squads/          — 8 squads funcionais
├── workspace/       — estado operacional (preenchido em uso)
└── .claude/         — skills, agentes e hooks
```

## Recursos

- `/jade-iniciar` — abre sessão com fila de pendências
- `/configurar-squad` — configura identidade do operador
- `/jade` — COO orchestrator
