# Aprendizados — Squad Copy

_Registre aqui os aprendizados acumulados do squad. Cada correção do operador deve virar uma entrada._

---

## Como usar este arquivo

- Cada aprendizado deve ter data, contexto, correção e regra nova
- Ler este arquivo antes de executar qualquer produção do squad
- Ao receber correção do operador → registrar aqui antes de prosseguir (Regra §5)

---

## Exemplo de entrada

**[YYYY-MM-DD] — Título do aprendizado**

Contexto: o que aconteceu, qual output foi produzido.
Correção: o que o operador corrigiu ou apontou.
Regra nova: como evitar reincidência.
