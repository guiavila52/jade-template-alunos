# Aprendizados — revisor-roteiro

_Registre aqui os aprendizados acumulados deste agente. Cada correção do operador deve virar uma entrada._

_Este arquivo é lido pelo agente antes de executar qualquer produção (Regra §5)._

---

## Como usar este arquivo

- Cada entrada deve ter data, contexto, correção e regra nova
- Ler antes de produzir qualquer output
- Ao receber correção do operador → registrar aqui imediatamente

---

## Exemplo de entrada

**[YYYY-MM-DD] — Título do aprendizado**

Contexto: o que aconteceu, qual output foi produzido.
Correção: o que o operador corrigiu ou apontou.
Regra nova: como evitar reincidência.
