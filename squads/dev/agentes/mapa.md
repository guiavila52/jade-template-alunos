# Mapa de Agentes — Squad Dev

- **desenvolvedor-frontend** — Desenvolve e mantém páginas web (Astro, HTML, CSS)
- **designer-revisor** — Revisão visual headless (Playwright) antes de publicar
- **designer-ui** — Design de interfaces e sistemas visuais
- **devops** — Deploy, infraestrutura, CI/CD
- **analista-qa** — Testes e qualidade antes de entregar
