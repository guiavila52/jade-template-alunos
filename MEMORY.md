# MEMORY — Memória persistente da Jade

> Este arquivo é gerado e atualizado automaticamente pela Jade entre sessões.
> NÃO editar manualmente. A Jade salva aqui aprendizados, decisões e estado operacional.

## Como funciona

Ao final de cada sessão importante, rode `/consolidar-sessao` — a Jade salva o que importa aqui.

## Estrutura (criada pela Jade conforme uso)

Seções são criadas dinamicamente conforme o squad evolui.
