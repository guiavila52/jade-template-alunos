# 05 — Audiência

Coloque aqui tudo que você sabe sobre sua audiência.

Sugestões:
- Perfil do cliente ideal (dores, desejos, objeções)
- Segmentos de público (se houver mais de um)
- Linguagem que ressoa com sua audiência
- Perguntas frequentes recebidas
- Depoimentos e provas sociais coletadas

> Dica: audiência bem mapeada = copy que converte, conteúdo que engaja.
