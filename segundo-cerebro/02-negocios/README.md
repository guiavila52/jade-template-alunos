# 02 — Negócios

Coloque aqui tudo sobre suas empresas, produtos e modelo de negócio.

Sugestões:
- Descrição de cada produto/serviço com preço
- Funil atual (topo, meio, fundo)
- Proposta de valor única
- Diferenciais vs concorrentes
- Métricas importantes (ticket médio, conversão, LTV)
- Metas financeiras

> Dica: um produto bem descrito aqui gera copy, criativos e páginas muito melhores.
