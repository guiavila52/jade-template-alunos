Use o MCP do Gmail:

## Regras

- Não narre. Não diga "vou buscar", "vou analisar". Execute e entregue o cockpit.
- Blocos livres são tão importantes quanto os compromissos — sempre calcule e mostre.
- Emails: só os que importam. Não liste inbox completo.
- Foco de hoje: uma opinião forte. Não "depende" — diz qual é a prioridade.
- Se alguma integração falhar, entregue o cockpit parcial com o que conseguiu e sinalize o que faltou.
